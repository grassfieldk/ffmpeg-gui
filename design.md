
# FFmpeg GUI

Windows 上で動作する FFmpeg の GUI ツール
複数の動画をドラッグドロップで追加して、よく使う変換パターンに沿って変換する


## 入力

- 動画のドラッグ＆ドロップ
- 追加できる動画は 1 つのみ


## 変換

次の項目を選べるようにする
各項目の初期値は先頭の動画に合わせる

- 解像度
  - 基本的には数値指定
  - 480p、720p、1080p と 縦/横 の組み合わせの系 6 パターンを用意し、選択すると数値が自動で入力される
- 動画ビットレート
- フレームレート形式（固定・可変）
- フレームレート
- 音声フォーマット
- 音声ビットレート
- 画質（CRF）

### プリセット

各項目を自動で決めるためのプリセット機能
これらのプリセットを選択すると、各項目の値が自動で設定される

- 編集用
  - H.264、固定フレームレート
- SNS 投稿用
  - 480p、CRF 28、固定 30fps


## 出力

- 入力ファイルと同じフォルダに `{input_file_name}_converted.{ext}` として保存
- 拡張子はすべて小文字で自動選択（ユーザーが変更できない）


## UI

- 画面遷移はなし、1 画面のみ（モーダル・ポップアップは可）
- コード量が少なくなるように最低限の装飾
- 各項目ごとにデフォルト値（元動画の値）へのリセットボタンが存在
- 画面上に実行ログを表示
